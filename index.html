Smoopyvet

Smoopyvet es un sistema medico veterinario. se centra en el manejo de historia clinica, control de vacunacion, guarderia y demas servicios.

Aplicacion movil vacunacion

En esta version encontraremos un control de vacunacion para nuestras mascotas. se encuentra disponible para dispositivos Android y Ios.

Aplicacion web control lechero

En esta version encontraremos un sistema para el control lechero por hatos para las fincas ganaderas. actualmente se encuentra en construccion.

Aplicacion guarderia canina pc

Esta version cuenta con el control de guarderia. inicialmente se encuentra disponible para windows. proximamente para linux y mac.

Aplicacion movil historia clinica

En esta version encontraremos un control de historia clinica. esta es una version beta. estara disponible para dispositivos Android y Ios.

Aplicacion web comercio electronico

En esta version encontraremos un sistema para tienda comercio electronico para venta de productos veterinarios.

Aplicacion web rastreo mascotas perdidas

En esta version encontraremos un sistema para rastreo mediante chip de mascotas perdidas.

Aplicacion seguimiento transporte mascotas

En esta version encontraremos un sistema de seguimiento para el transporte de nuestras mascotas de la guarderia a la casa.
